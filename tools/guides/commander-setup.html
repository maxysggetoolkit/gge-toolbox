🎖️ Attack Commander Setup

Intermediate Equipment Gems Commander ~6 min read

A commander is the sum of its gear. Five equipment slots, four gem sockets, and a set bonus — pointed at the positions and unit types you actually attack with.

Read this first — relic gear is the modern meta. Most serious players now run relic equipment over fixed event sets. The slot conventions below describe set/event gear; relic gear largely ignores them. Jump to Relic equipment for why.

The slots (set & event gear)

A commander carries five pieces. On set and event gear, each slot tends to lean toward certain stats — a useful starting framework:

SlotTends to carry
WeaponCombat strength, melee/ranged attack.
HelmetCourtyard, wall protection, speed.
Armor (chest)Front, gate, moat.
ArtifactFlank, moat, resources/utility.
HeroThe commander's signature — big position bonuses.

These "tends to carry" rules are conventions, not laws — and relic gear breaks them on purpose (see below).

What stats actually matter on attack

Damage on offence comes from position bonuses (front / flank / courtyard) and unit-type bonuses (melee / ranged), plus flat combat strength. Aim to cover:

  • Courtyard — the final fight; almost always worth stacking.
  • Front and Flank — match where you commit your units.
  • Melee and Ranged attack — for whichever unit type you field.
  • Combat strength — a clean multiplier on top of the above.
Don't spread thin. A commander that's +60% on one position beats one that's +20% on three. Decide what this commander does (e.g. ranged courtyard hitter) and stack that.

Gems (the four sockets)

Gems fine-tune the build. A common, balanced split across the four sockets:

  • Socket 1 — flank / front / courtyard.
  • Socket 2 — courtyard / ranged / melee.
  • Socket 3 — flank / front / courtyard.
  • Socket 4 — courtyard / ranged / flank / front.

Wall, gate and moat lines exist too — useful on a defensive build, mostly wasted on a pure attacker. Spread the defensive lines evenly if you do run them.

Useless on an attacker: glory and fire bonuses are dead stats for offence. If a piece's only "bonus" lines are those, it isn't an attack piece.

Relic equipment — the current meta

Set and event gear has fixed stats. Relic gear is different, and it's what most strong players now build around. Each relic piece (armor, weapon, helmet, artifact, hero — and relic gems) carries relic effects that roll within a value range, and you can enchant them to push those values higher. You're not collecting a set — you're rolling and upgrading the exact stats you want.

That's why relic gear ignores the "tends to carry" conventions above: any relic slot can roll from its effect pool, so a relic helmet can be your big courtyard piece, your weapon, or your flank piece, depending on what you rolled. You build toward a target stat line rather than a fixed slot layout.

Relic gear is target-specific

Relic effects come in flavours tied to what you're attacking:

  • Castle lords (PvP) — for hitting other players.
  • NPC targets (PvE) — robber barons and general PvE.
  • Nomad, Samurai, Glory (alien) invasion, Berimond — event-specific.

So a "best" relic commander depends on the target. Many players keep several relic commanders, each rolled for one job, and swap between them — rather than one do-everything set.

Practical takeaway: a well-rolled, enchanted relic commander usually beats a fixed set for the job it's built for. Use fixed sets (and the Rift sets below) where the set bonus itself is the point — like the Rift Raid event — and relic gear for everyday PvP/PvE hitting.

Set bonuses (where sets still win)

Wearing several pieces of the same set unlocks set bonuses at 2, 3, 4… pieces. When the set bonus is strong and event-specific — Rift sets are the clearest case — a full set beats mixed relic pieces, because the thresholds stack. Count the set bonuses you'd light up, not just each piece's own stats.
