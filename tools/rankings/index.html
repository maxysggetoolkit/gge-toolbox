📊 Player & Alliance Rankings

The game's own live highscore lists — might, honour, legendary level and achievement points — per server. Pick a category and page through, or type a name to jump straight to that entry's rank.

About the data: these are the real in-game highscore lists, read live from the Goodgame Empire servers via danadum's empire-api — the same source as the VIP Live Event Rankings. They match the game exactly (AU1 might is topped by Hards161 at ~495M), so they don't suffer the gaps third-party crawl indexes have. Might lists are split by level bracket; Level 70 holds the big accounts.
