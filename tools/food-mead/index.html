🍖 Food & Mead Production

Tally the food, beef, honey and mead production your decorations and build items add up to. Set how many of each you run and compare builds — handy when you're deciding which Exalted Green decos or granary items earn their tiles.

Reading the numbers: these are flat hourly bonuses (food/beef/honey per hour; mead is the total production bonus). There's no decoration cap on production the way there is on defence — your real ceiling is consumption: troops eat food, sovereign troops drink mead. Pair this with the Food and Mead optimisation guides. Values from the game data cache (research % boosts and base building output not included).
